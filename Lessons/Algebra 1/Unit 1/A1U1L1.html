Football and Integers

The football field is a great place to see how integers are used in real life. For example:

A quarterback throws a 37-yard pass from his team's 8-yard line. During the next play, the offensive team commits a holding penalty.

This penalty places the ball back 10 yards from the previous play's completion. The diagram shows how the throw is in a positive direction and the penalty is in a negative direction.

The diagram indicates the starting and ending points of the ball for this play:

8-yard line (starting point) + 37-yard throw = 45-yard line
(This is where the throw finishes.)

Mathematically speaking:

8+37=45

45-yard line – 10-yard penalty = 35-yard line
(This is where the ball ends up.)

Mathematically speaking:

45−10=35

45-10=35

Number Lines

A number line can be used to add or subtract Rational numbers. One may draw an arrow to the right to add a positive number and to the left to add a negative number.

For example:

3.5+(-4.5)=?

You should be thinking, "Right 3.5 spaces, and then left 4.5 spaces.

EXAMPLE 1

\[ (- {3 \over 4}) + -{1 \over 8} = ? \]

\[ - {6 \over 8} + {1 \over 8} \]

LCD (Lowest common denominator) is 8. Find the equivalent fraction for 3/4.

\[ {-6+1 \over 8} \]

Keep the denominator. Perform top operation. The signs are different. Subtract and keep the sign of the number with the larger Absolute value.
\[ |-6| = 6, |1| = 1, 6 \geq 1 \]

\[ - {5 \over 8} \]

Therefore, the sign is negative.

Rules for adding numbers:

A Positive value plus a Positive value is a Positive value → Add the absolute value of the numbers and keep the same sign.
A Negative value plus a Negative value is a Negative value → Add the absolute value of the numbers and keep the same sign.
A Positive value plus a Negative value could be positive or negative → Subtract the absolute value of the numbers and keep the sign of the

Note: Anytime you come across subtraction, you can rewrite the subtraction using the Additive Inverse property which essentially lets you "add the opposite." For instance:

ORIGINAL EXPRESSION

2 - 1

2 minus 1

3 - 7

3 minus 7

5 - 3

5 minus 3

-6 - (-4)

-6 minus -4

REWRITTEN EXPRESSION

2 + (-1)

2 plus -1

3 + (-7)

3 plus -7

5 + (-3)

5 plus -3

-6 + (-4)

-6 plus -4

With decimals, the same rules apply as with integers.

EXAMPLE 2

\[ -7.6 - (3.94) = ? \]

\[ -7.6 + (-3.94) = ? \]

Rewrite the subtraction as an addition. This is an application of Additive inverse.

\[ \displaylines {-7.6 \\ -3.94 \over} \]

Line up the decimals and add.

\[ - 11.54 \]

Keep the sign, since both numbers have the same sign.

EXAMPLE 3

\[ ({ -1 \over 2}) - ({ -3 \over 4}) = ? \]

\[ ({ -1 \over 2}) + ({ 3 \over 4}) = ? \]

Rewrite problem applying additive inverse.

\[ {-2 \over 4} + {3 \over 4} \]

Lowest Common Denominator [LCD] is 4. Find an equivalent fraction for 1/2.

\[ {-2 + 3 \over 4} \]

Keep the denominator and perform top operation. The signs are different. Subtract and keep the sign of the number with the larger absolute value.

\[ {1 \over 4} \]

Therefore, the sign is positive.
