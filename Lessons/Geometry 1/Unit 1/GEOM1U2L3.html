What You Should Know

Before you begin, you should know:

  • How to recognize adjacent angles, vertical angles, complementary angles, supplementary angles, and linear pairs
  • How to find the measurement (in degrees) of any missing angles involving vertical angles, complementary angles, supplementary angles, and linear pairs
  • How to solve basic Algebra I equations
Lesson Expectations

By the end of this lesson, you should be able to:

  • Write and solve equations involving adjacent, vertical, complementary, and supplementary angles
  • Complete proof of angle relationships

Given the following diagram…

A few different things can be asked of you, one possible question is to find the value of x, another is to find the measure of a single angle or all the angles….

What you need to do to handle these types of problems is to first find what angle relationship exists. Are the angles vertical, complementary, supplementary? Are they part of an angle bisector, etc.?

In this case, the angles are Vertical angles. Use the definition of this relationship (in this case, the definition says that vertical angles are congruent angles) to write an equation.

Since vertical angles are congruent,
5x−8=3x+10
Now solve for x.

Using the knowledge you learned in Algebra I, you will get
x=9

If the question asks you to find the value of x, then you are done; but if the question asks you to find the angle measure, then you take the value of x and substitute it back into the expressions.
