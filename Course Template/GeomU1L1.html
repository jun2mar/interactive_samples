Let's continue with the following terms:

1. Midpoint: halfway between two points (a part in the middle).

A. On a number line

If we want to find the midpoint between a segment AB on a number line, we just count how many units there are from point A to point B.

We can actually subtract the two values and find the absolute value.

/(-5)-(5)/ = /-10/

And the absolute value of -10 is 10, that gives us the distance between point A and point B To find the midpoint we have to divide that result by 2.

-5+5=0
On this occasion, the midpoint is 0.
